# 🎟️ Concert Ticket Batch Generator 

这是一个基于 Python Pillow 库开发的门票批量生成工具，专为同济大学学生民乐团 2026 “银幕如歌，丝竹弄影”专场音乐会设计。

## 📖 项目背后的故事 (为什么会有这个程序？)

这是一个关于“程序员震怒”的故事：

2026年4月某日上午，我们联系了打印店，提供了门票模板和详细的座位分布表，要求下午两点半前完成打印。

结果打印店回复**要一个一个手敲座号，得干一下午，第二天才能完成。**，甚至说他们经常做排票的工作，都是这么干的。

听到“手敲”二字，我顿时震怒。在 2026 年的今天，这种原始且低效的工作方式简直是对效率的亵渎。于是，我花了 50 分钟“怒搓”了这个程序。

**结果：** 程序运行不到 1 分钟，全场八百余张票的渲染图就全部生成完毕。

本想把代码共享给打印店换点折扣，转念一想，他们要是能跑通 Python 环境，大概也不会在那里手敲座号了。为了乐团的效率，先传到 GitHub 备份。未来如果有空，我会考虑将其封装成带 GUI 的软件供普通用户直接使用。

## 🚀 功能特性

- **自动化排座**：根据预设的区域（A/B/C/D区）和行列规则，自动生成全场座位数据。
- **批量渲染**：读取 `picture.png` 模板，在指定坐标自动填入“区”、“排”、“座”信息。
- **高效率**：50 分钟开发，秒级生成数百张门票图片。
- **自定义配置**：支持自定义字体、坐标偏移和文本颜色。
